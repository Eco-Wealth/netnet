# Evaluation Rubric

Total score: 100.

## Objective Coverage (0-30)
- 30: Every requested deliverable completed.
- 15: Partial completion.
- 0: Core objective missed.

## Acceptance Gates (0-40)
- 40: All required tests and netnet gates pass with evidence.
- 20: Partial evidence or mixed pass/fail.
- 0: No acceptance proof.

## Scope Discipline (0-15)
- 15: No out-of-scope changes.
- 8: Minor unapproved drift.
- 0: Major scope expansion.

## Artifact Quality (0-15)
- 15: Work order and run report are valid and machine-readable.
- 8: Artifacts exist with minor contract gaps.
- 0: Missing or invalid artifacts.

## Thresholds
- 85-100: production-ready iteration.
- 70-84: acceptable with cleanup pass.
- <70: rework required.
